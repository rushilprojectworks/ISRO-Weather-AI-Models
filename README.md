# 🛰️ ISRO Internship: AI Models for Weather Datasets
**Location:** SDSC-SHAR, Sriharikota | Mission Computers Division

### **Overview**
This project identifies suitable AI models for predicting weather patterns using historical data (2009–2016).

### **Key Findings**
* **Best Model:** Long Short-Term Memory (LSTM) Networks
* **Accuracy:** Achieved an **R² of 0.90** and an **MAE of 1.4**
* **Applications:** Real-time processing and mission planning support at ISRO

### **Project Structure**
* `ISRO INTERN REPORT BY RUSHIL.docx`: Official internship report
* `weather_models.py`: Python implementation of ARIMA and LSTM models
